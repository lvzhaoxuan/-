### 使用方法：
首先你需要下载nodejs,百度，设置下环境变量
接着pull本项目，然后执行
```
npm run serve
```
提示:
```

  - Local:   xxx.xxxx.xxx
  - Network: xxx.xxx.xxxx
```
则项目启动成功,
打开浏览器，输入下方任意一个地址就可以访问本项目了，一般用localhost就行
